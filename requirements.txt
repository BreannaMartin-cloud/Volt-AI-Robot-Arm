# VOLT robot - Python dependencies (Raspberry Pi 4, Raspberry Pi OS)
#
# Install with:
#   pip3 install -r requirements.txt --break-system-packages
#
# opencv, numpy, and picamera2 MUST come from apt on a Pi (Picamera2 is
# apt-only, and a pip opencv alongside apt's python3-opencv is exactly the
# "two versions of cv2" conflict that broke the camera before):
#   sudo apt install python3-opencv python3-numpy python3-picamera2
#
# Do NOT pip install opencv-python or numpy on the Pi. mediapipe is also
# left out on purpose - pip mediapipe drags in its own opencv wheel and
# recreates the same conflict; vision.py falls back to OpenCV's Haar
# face detector automatically when mediapipe is absent.
